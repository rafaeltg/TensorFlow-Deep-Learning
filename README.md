# Deep Learning algorithms with Keras library

Deep Learning algorithms implemented using Keras library and Python.

### Available Models:
* Autoencoder
* Denoising Autoencoder
* Deep Autoencoder
* Stacked Autoencoder
* Stacked Denoising Autoencoder
* MultiLayer Perceptron
